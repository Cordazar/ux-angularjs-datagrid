The purpose of this application is to test memory usage of the datagrid. By going from page to page to validate that the memory footprint is cleared.

Addons that are ready to apply to your grid.
  • Desktop
    • disableHoverWhileScrolling - in desktop applications with hovers in the scrolling performance can decrease on scrolling dom. This will automatically disable hovers while scrolling. see http://www.thecssninja.com/javascript/pointer-events-60fps for more information.
  • Touch
    • IOS
      • iosScrollFriction - This will only work on an ios device. Android does this by default.
    • infiniteScroll - Show a spinner as the grid scrolls to the bottom and then load more data.
    • statsModel
