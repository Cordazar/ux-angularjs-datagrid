Collapsible Groups View Docs

collapsibleGroups works on the concept of starting out with rows already exposed. It doesn't actually change that the grid renders, but just does a display none on that dom. It is recommended to use ExpandableGroups instead because it will be more performant with a large data set. collapsibleGroups is still digesting all of the rows that are collapsed. It is really the early version of expandableGroups.
